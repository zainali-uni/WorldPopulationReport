# Product Backlog

## High Priority

### US1 – View All Countries by Population
As a user, I want to view all countries ordered by population from largest to smallest so that I can understand global population distribution.

### US2 – View All Cities by Population
As a user, I want to view all cities ordered by population so that I can compare city sizes worldwide.

### US3 – View Capital Cities Report
As a user, I want to view all capital cities ordered by population.

### US4 – View Top N Populated Countries
As a user, I want to view the top N most populated countries, where N is provided by the user.

---

## Medium Priority

### US5 – View Top N Populated Cities
As a user, I want to view the top N most populated cities.

### US6 – Population Breakdown by Continent
As a user, I want to view population grouped by continent.

### US7 – Population Breakdown by Region
As a user, I want to view population grouped by region.

### US8 – Language Speaker Statistics
As a user, I want to view statistics of major language speakers worldwide.

---

## Low Priority

### US9 – Improve Output Formatting
As a user, I want reports to be well formatted for better readability.

### US10 – Advanced Error Handling
As a developer, I want advanced error handling to ensure system stability.
